## 🚀 How to Run Locally

1. **Clone this repository**
   ```bash
   git clone https://github.com/macsyne/my-portfolio.git
   ```

2. **Go into the project folder**
   ```bash
   cd my-portfolio
   ```

3. **Install the requirement**
   ```bash
   pip install streamlit
   ```

4. **Run the app**
   ```bash
   streamlit run home.py
   ```
